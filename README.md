# TG Group/User ID Bot

Простой бот на Python, который по команде `/start` отправляет:

- в личном чате: ID пользователя;
- в группе или супергруппе: ID этой группы.

## Запуск

1. Установите зависимости:
   ```bash
   python -m venv .venv
   .\.venv\Scripts\activate
   pip install -r requirements.txt
   ```
2. Создайте файл `.env`, скопировав `.env.example`, и впишите реальный токен телеграм-бота:
   ```
   BOT_TOKEN=ваш_токен
   ```
3. Запустите бота:
   ```bash
   python bot.py
   ```

## Использование

- Напишите боту в личном чате `/start`, чтобы получить свой ID.
- Добавьте бота в группу, дайте ему право читать сообщения и отправьте `/start`, чтобы узнать ID группы.
